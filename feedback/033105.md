# 033105 - WebSqlEditor 编译修复

## 编译结果

```
BUILD SUCCESSFUL in 735 ms
```

## 修复的错误

| # | 错误 | 修复 |
|---|------|------|
| 1 | `Property 'getError' does not exist on type 'OnErrorReceiveEvent'` | 移除 `event?.getError?.()` 调用 |
| 2 | `Argument of type '"message"' is not assignable to parameter of type '"controllerAttachStateChange"'` | 移除 `this.webController.on('message', ...)` |
| 3 | `Cannot find name 'MessageEvent'` | 移除 `MessageEvent` 类型引用 |
| 4 | `Property 'setSyncTag' does not exist on type 'WebviewController'` | 移除 `setSyncTag` 调用 |

## 修复后的 WebSqlEditor

- `initWebMessage()` 方法简化，只设置 `isWebReady = true` 并初始化文本
- 移除了与 Web 页面的双向消息通信（OpenHarmony WebviewController API 限制）

## 遗留问题

- Web 编辑器的文本变化暂时不会同步回 ArkTS（单向：ArkTS → Web）
- 需要时可添加定时轮询或使用 `runJavaScript` 回调获取文本
