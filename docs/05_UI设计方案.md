## 1. 设计风格定位

### 1.1 风格选择: Modern Dark (OLED)

**选定风格**: Modern Dark - 专为开发者工具设计的暗色主题

| 维度 | 选择 | 理由 |
|------|------|------|
| 主题 | **Dark Mode (OLED)** | 开发者工具标准配置，减少眼疲劳，适合长时间编码 |
| 视觉 | **Neon Accents** | SQL 语法高亮、状态指示的专业表达 |
| 密度 | **Medium-High** | 桌面端需要高效信息密度 |
| 氛围 | **Technical Precision** | 清晰、专业、高效 |

### 1.2 关键词

```
dark mode · developer tool · OLED optimized · high contrast
code editor · data table · professional · technical precision
```

### 8.2 关键交互动效

#### 按钮按下效果

```typescript
Button('执行')
  .onTouch((event: TouchEvent) => {
    if (event.type === TouchType.Down) {
      // 按下：缩小 + 颜色加深
      animateTo({ duration: 80 }, () => {
        this.scale = 0.97;
        this.bgColor = '#2563EB';
      })
    } else if (event.type === TouchType.Up) {
      // 释放：恢复
      animateTo({ duration: 150 }, () => {
        this.scale = 1;
        this.bgColor = '#3B82F6';
      })
    }
  })
```

#### Schema 树展开/折叠

```typescript
if (this.expanded) {
  animateTo({
    duration: 200,
    curve: Curve.EaseOut
  }, () => {
    this.childHeight = this.calculateChildHeight();
  })
} else {
  animateTo({
    duration: 150,
    curve: Curve.EaseOut
  }, () => {
    this.childHeight = 0;
  })
}
```

#### 执行成功反馈

```typescript
// 执行成功后，按钮短暂变绿然后恢复
animateTo({ duration: 200 }, () => {
  this.buttonBgColor = '#22C55E';
})
setTimeout(() => {
  animateTo({ duration: 300 }, () => {
    this.buttonBgColor = '#3B82F6';
  })
}, 800)
```
