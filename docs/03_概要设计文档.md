## 4. 模块设计

### 4.1 RDB Store API 模块 【已修订】

**模块职责**: 使用 OpenHarmony 内置 RDB Store API 操作 SQLite 数据库

**核心接口**:

```typescript
// @ohos.data.rdb 核心接口
interface RdbStore {
  // 数据库操作
  open(config: StoreConfig): Promise<RdbStore>;
  close(): void;

  // 数据操作
  insert(table: string, values: ValuesBucket): Promise<number>;
  update(table: string, values: ValuesBucket, predicates: RdbPredicates): Promise<number>;
  delete(predicates: RdbPredicates): Promise<number>;
  query(predicates: RdbPredicates, columns: string[]): Promise<ResultSet>;

  // SQL 执行（注意：executeSql 无法直接返回 ResultSet）
  executeSql(sql: string, args: string[]): Promise<void>;
}
```

> **重要说明**：RDB Store API 的 executeSql() 方法返回 `Promise<void>`，不支持直接获取查询结果。SELECT 查询需使用 `query()` + `RdbPredicates` 方法。

### 4.1 FileTransferService 模块 【新增】

**模块职责**: 处理文件的导入（外部→沙箱）和导出（沙箱→外部），解决 HarmonyOS 沙箱隔离问题

**核心接口**:

```typescript
// services/FileTransferService.ets
class FileTransferService {
    /**
     * 获取应用沙箱内的数据库存储路径
     */
    getSandboxDatabasePath(): string;

    /**
     * 将外部数据库文件拷贝到沙箱
     * @param sourcePath 用户选择的原始文件路径
     * @returns 沙箱内的新路径
     */
    copyToSandbox(sourcePath: string): Promise<string>;

    /**
     * 将沙箱内的数据库导出到外部位置
     * @param sandboxPath 沙箱内文件路径
     * @param destPath 用户选择的导出目标路径
     */
    exportToOriginal(sandboxPath: string, destPath: string): Promise<void>;

    /**
     * 删除沙箱内的数据库副本
     * @param sandboxPath 沙箱内文件路径
     */
    deleteSandboxCopy(sandboxPath: string): Promise<void>;

    /**
     * 检查沙箱内是否存在指定文件
     */
    existsInSandbox(fileName: string): boolean;
}
```

**文件操作流程**：

```
[打开数据库]
用户选择文件 → DocumentViewPicker → copyToSandbox() → 沙箱副本路径

[保存/导出数据库]
沙箱修改 → exportToOriginal() → 用户选择位置 → 写回原位置

[关闭数据库]
沙箱副本保留（供下次快速打开）或删除（用户可选）
```

### 4.2 NAPI SQLite 管理器模块

**模块职责**: 高性能 SQL 查询执行和结果处理

**核心接口**:

```cpp
// query_engine.h
class QueryEngine {
public:
    QueryEngine(sqlite3* db);
    ~QueryEngine();

    // 查询执行
    QueryResult execute(const std::string& sql, const std::vector<Value>& args);
    QueryResult rawQuery(const std::string& sql);

    // 查询计划
    std::string explain(const std::string& sql);

    // 事务管理
    bool begin();
    bool commit();
    bool rollback();

private:
    sqlite3* m_db;
    bool inTransaction;
};
```

### 4.3 ArkTS 数据库服务模块 【已修订】

**模块职责**: 封装数据库操作，为 UI 层提供统一接口

**核心接口**:

```typescript
// services/DatabaseService.ets
class DatabaseService {
    // 数据库管理
    init(context: UIAbilityContext): Promise<void>
    openDatabase(dbPath: string): Promise<void>
    closeDatabase(): Promise<void>
    isDatabaseOpen(): boolean

    // Schema 操作
    getTables(): Promise<TableInfo[]>
    getColumns(tableName: string): Promise<ColumnInfo[]>
    getIndexes(tableName?: string): Promise<IndexInfo[]>

    // 数据操作（使用 predicate-based API）
    select(table: string, columns?: string[]): Promise<QueryResult>
    insert(table: string, data: Record<string, string | number | null>): Promise<number>

    // SQL 执行
    executeSql(sql: string, args?: string[]): Promise<QueryResult>

    // 事务
    beginTransaction(): Promise<void>
    commit(): Promise<void>
    rollback(): Promise<void>
}
```

> **实现说明**：由于 executeSql 无法返回 ResultSet，SELECT 查询使用 `query()` + `RdbPredicates` 实现。

## 7. 错误处理

### 7.1 错误分类

| 错误类型 | 错误码范围 | 说明 |
|----------|------------|------|
| 数据库错误 | 1000-1999 | SQLite 相关错误 |
| Schema 错误 | 2000-2999 | Schema 操作相关错误 |
| SQL 执行错误 | 3000-3999 | SQL 语法/执行错误 |
| 文件系统错误 | 4000-4999 | 文件读写相关错误 |
| 系统错误 | 5000-5999 | 系统级错误 |

## 8. 安全性设计

### 8.1 安全措施

| 安全措施 | 描述 | 实现方式 |
|----------|------|----------|
| 文件访问控制 | 仅允许访问用户明确选择的数据库文件 | 文件选择器确认 |
| SQL 注入防护 | 防止恶意 SQL 语句执行 | 参数化查询 |
| 敏感信息保护 | 不在日志中记录敏感数据 | 日志脱敏 |
| 进程隔离 | 防止数据库文件损坏影响系统 | 沙箱机制 |

## 9. 性能设计

### 9.1 性能优化策略

| 优化项 | 策略 | 预期效果 |
|--------|------|----------|
| 数据加载 | 分页加载，避免一次性加载大量数据 | 减少内存占用 |
| UI 渲染 | 懒加载 + 虚拟列表 | 提升列表滚动流畅度 |
| SQL 执行 | 异步执行，避免阻塞 UI 线程 | 提升响应性 |
| 图片/资源 | 按需加载 | 减少初始加载时间 |

### 9.2 资源限制

| 资源类型 | 限制值 | 说明 |
|----------|--------|------|
| 单次查询结果集 | 10000 行 | 超出自动分页 |
| 单个单元格数据 | 2 MB | 超出显示截断提示 |
| 数据库文件大小 | 受文件系统限制 | 无应用层限制 |
| 并发查询 | 1 个 | 当前版本单连接 |
